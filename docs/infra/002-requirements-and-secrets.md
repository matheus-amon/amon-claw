# Requisitos Técnicos e Segredos

## Requisitos de Ambiente
- Conta na AWS (com permissões Admin).
- Conta no MongoDB Atlas (Cluster M0 criado).
- Repositório no GitHub.

## Segredos (GitHub/SSM)
| Nome | Descrição |
| :--- | :--- |
| `AWS_ACCESS_KEY_ID` | Acesso à AWS. |
| `AWS_SECRET_ACCESS_KEY` | Segredo da AWS. |
| `MONGODB_URI` | URI do Atlas. |
| `OPENROUTER_API_KEY` | Chave da IA. |
| `EVOLUTION_API_KEY` | Chave do WhatsApp. |
| `EVOLUTION_API_URL` | URL do WhatsApp. |
